Apa itu Web Desain?

Web desain adalah cara membuat tampilan sebuah website supaya enak dilihat dan mudah dipakai. Tujuannya supaya orang yang membuka website gampang menemukan apa yang mereka cari, betah lama-lama di website itu, dan percaya dengan informasi atau produk yang ditawarkan. Desain web juga dibuat supaya website bisa dibuka dengan baik di komputer, HP, atau tablet, sehingga semua orang bisa mengaksesnya dengan nyaman.

Apa itu Coding?

Coding adalah cara kita memberi perintah ke komputer supaya dia melakukan sesuatu. Bayangin kamu kasih tahu komputer langkah-langkah yang harus dikerjakan, misalnya buat tampilkan tulisan, buat tombol, atau jalankan aplikasi. Perintah-perintah ini ditulis dengan bahasa khusus yang dimengerti komputer, namanya bahasa pemrograman. Jadi, coding itu seperti membuat resep atau petunjuk agar komputer tahu apa yang harus dilakukan.

Manfaat Belajar Coding

Belajar coding punya banyak manfaat, terutama buat kamu yang ingin lebih paham teknologi dan bisa membuat sesuatu sendiri di komputer atau internet. Dengan belajar coding, kamu bisa membuat website, aplikasi, game, atau bahkan otomatisasi pekerjaan supaya lebih cepat dan mudah. Selain itu, kemampuan coding juga bisa membuka peluang kerja yang lebih banyak dan gaji yang lebih baik karena sekarang banyak perusahaan butuh orang yang bisa buat dan perbaiki program komputer. Selain itu, belajar coding juga melatih cara berpikir logis dan memecahkan masalah dengan lebih teratur. Jadi, belajar coding itu seperti punya kemampuan super untuk menciptakan hal baru dengan komputer

Peluang Kerja

Peluang kerja di bidang coding sangat besar dan terus berkembang seiring kemajuan teknologi. Hampir semua perusahaan sekarang butuh programmer atau developer untuk membuat website, aplikasi, sistem, dan berbagai solusi digital. Mulai dari startup kecil sampai perusahaan besar, bahkan pemerintah pun membutuhkan orang yang bisa coding. Selain itu, kamu bisa kerja di berbagai bidang seperti teknologi informasi, keuangan, kesehatan, hiburan, dan banyak lagi. Selain kerja di perusahaan, kamu juga bisa jadi freelancer atau buka jasa pembuatan website dan aplikasi sendiri. Jadi, kemampuan coding membuka banyak jalan karier dengan gaji yang kompetitif dan peluang untuk terus belajar hal baru.

Belajar HTML

HTML adalah bahasa dasar untuk membuat halaman web yang memberi tahu browser apa yang harus ditampilkan.

Contoh Kode HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertamaku</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertamaku.</p>
  </body>
</html>

Penjelasan

  • <html> = pembuka halaman
  • <head> = bagian judul dan info lain
  • <body> = isi utama yang terlihat
  • <h1> = judul besar
  • <p> = paragraf

Belajar CSS

CSS digunakan untuk mempercantik tampilan halaman HTML dengan warna, ukuran, dan tata letak.

Struktur CSS

CSS terdiri dari selector diikuti properti dan nilai:

selector {
  property: value;
}

Contoh Kode CSS

h1 {
  color: blue;
  text-align: center;
}

p {
  font-size: 18px;
  color: gray;
}

Penjelasan

  • color = warna teks
  • text-align = posisi teks
  • font-size = ukuran huruf

Cara Menggunakan CSS

  • Inline CSS: langsung di tag HTML
  • Internal CSS: dalam tag <style>
  • Eksternal CSS: file .css terpisah

Biasanya menggunakan eksternal karena lebih rapi.
