â“ Materi 1: Apa Itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif dan dinamis. Dengan JavaScript, pengembang dapat menambahkan fitur seperti animasi, validasi formulir, dan respon terhadap tindakan pengguna tanpa harus memuat ulang halaman. Bahasa ini berjalan langsung di browser dan sangat penting dalam pengembangan web modern.

ğŸ“Œ Materi 2: KEGUNAAN JavaScript

JavaScript digunakan untuk menambahkan interaktivitas pada halaman web, seperti membuat tombol yang merespons klik, menampilkan pesan pop-up, membuat animasi, hingga memperbarui konten secara otomatis tanpa perlu memuat ulang halaman. Selain itu, JavaScript juga digunakan untuk validasi formulir, pengambilan data dari server (AJAX), dan pengembangan aplikasi web modern seperti game online atau media sosial. Dengan JavaScript, pengalaman pengguna menjadi lebih dinamis, cepat, dan menarik.

  }

</body>

ğŸ§½ 5. Manipulasi DOM (Document Object Model)

Manipulasi DOM (Document Object Model) adalah proses mengubah struktur, konten, atau gaya elemen HTML secara dinamis menggunakan JavaScript. DOM merepresentasikan halaman web sebagai pohon elemen yang bisa diakses dan dimodifikasi oleh kode JavaScript. Dengan manipulasi DOM, kita bisa menambah, mengubah, atau menghapus elemen halaman tanpa harus memuat ulang halaman.


<!DOCTYPE html>
<html>
<head>
  <title>Manipulasi DOM</title>
</head>
<body>

<h2 id="judul">Judul Awal</h2>
<button onclick="ubahJudul()">Ubah Judul</button>

<script>
  function ubahJudul() {
    // Mengakses elemen dengan id 'judul'
    const elemenJudul = document.getElementById('judul');

    // Mengubah teks dalam elemen tersebut
    elemenJudul.innerText = "Judul sudah diubah!";

    // Mengubah gaya warna teks menjadi merah
    elemenJudul.style.color = "red";
  }
</script>

</body>
</html>
