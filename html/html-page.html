📘 Materi Dasar HTML

❓ Materi 1: Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa markah standar yang digunakan untuk membuat dan menyusun halamanweb. HTML berfungsi untuk menentukan struktur dasar sebuah halaman, seperti judul, paragraf, gambar, tautan, tabel, dan elemen-elemen lainnya agar dapat ditampilkan dengan baik di browser. HTML tidak mengatur tampilan visual secara langsung, tetapi lebih fokus pada kerangka dan makna konten. Sebagai contoh, kode HTML sederhana dapat menampilkan judul dan paragraf menggunakan tag seperti <h1> dan <p>. HTML juga memungkinkan penyematan media seperti gambar dan video, serta membuat tautan antar halaman(hyperlink), sehingga menjadi fondasi utama dari semua situs web.

🧱 Materi 2: Struktur Dasar HTML

HTML adalah bahasa standar untuk membuat halaman web. Dengan HTML, kita bisa menyusun teks, gambar, video, dan elemen lainnya.

Penting: Setiap elemen HTML memiliki struktur pembuka dan penutup!

Kunjungi sumber belajar terbaik di YouTube WPU.

Contoh Kode HTML


<!DOCTYPE html> 
<html>
  <head>
    <title>  Judul Halaman </title>
  </head>
  <body>
    <h1> Halo Dunia! </h1>
    <p> Ini paragraf pertama saya. </p>
  </body>
</html>
coba code

Penjelasan

  • <!DOCTYPE html> = Menandakan bahwa ini adalah dokumen HTML5
  • <html> = Pembungkus seluruh dokumen HTML
  • <head> = Berisi informasi untuk browser (judul, meta, link CSS, dll)
  • <title> = Menentukan judul tab browser
  • <body> = Tempat konten utama ditampilkan (tulisan, gambar, tombol, dll)
  • <h1> = Heading/judul besar
  • <p> = untuk membuat Paragraf

🧾 Materi 3: Elemen Teks dalam HTML

Elemen teks dalam HTML adalah tag untuk menampilkan dan mengatur teks, seperti <h1>–<h6> untuk judul, <p> untuk paragraf, <strong> untuk teks tebal, <em> untuk teks miring, dan <br> untuk pindah baris. Elemen ini membantu membuat teks lebih terstruktur dan mudah dibaca.

🔤 1. Heading (Judul)

HTML memiliki 6 tingkatan heading yang berfungsi untuk memberi struktur dan hierarki pada judul atau subjudul di halaman web. Tag heading terdiri dari <h1> hingga <h6>, di mana <h1> adalah tingkat tertinggi (paling penting) dan <h6> paling rendah. Heading membantu pengguna memahami urutan informasi dan memudahkan mesin pencari mengindeks konten dengan benar. 


<h1> Judul Besar </h1> 
<h2> Sub Judul </h2> 
<h3> Judul Kecil </h3> 
<h4> Judul Lebih Kecil </h4> 
<h5> Judul Sangat Kecil </h5> 
<h6> Paling Kecil </h6>

📄 2. Paragraf

Fungsi paragraf dalam HTML adalah untuk menampilkan teks dalam bentuk paragraf menggunakan tag <p>. Tag ini memisahkan blok teks agar lebih rapi, terstruktur, dan mudah dibaca oleh pengguna. Setiap paragraf akan otomatis diberi jarak oleh browser untuk membedakan satu paragraf dengan yang lain. 


<p> Ini adalah sebuah paragraf.</p> 
<p> Paragraf lainnya.</p>
coba code

💪 3. Teks Tebal dan Miring

Fungsi teks tebal dan miring dalam HTML adalah untuk memberi penekanan atau penegasan pada bagian tertentu dari teks. Teks tebal dibuat dengan tag <strong> atau <b>, dan digunakan untuk menandai informasi penting. Teks miring dibuat dengan tag <em> atau <i>, dan digunakan untuk menekankan atau menggarisbawahi makna kata dalam konteks kalimat. Selain memberi makna visual, tag <strong> dan <em> juga dikenali oleh mesin pencari dan pembaca layar, sehingga penting untuk aksesibilitas. 


<p> Ini adalah <strong> teks penting </strong>.</p>
<p> Ini adalah <em> teks miring </em>.</p>

🔗 4. Link (Tautan)

Link atau tautan dalam HTML digunakan untuk menghubungkan satu halaman web dengan halaman lain, baik di dalam situs yang sama maupun ke situs lain, menggunakan tag <a>. Atribut href di dalam tag tersebut menentukan alamat tujuan tautan, dan teks di antara tag <a> akan muncul sebagai teks yang bisa diklik. Tautan ini penting untuk navigasi dan mempermudah pengguna menjelajahi konten di web.


<a> href="https://www.google.com">Kunjungi Google</a>

📌 5. Garis Horizontal & Baris Baru

Fungsi garis horizontal dan baris baru dalam HTML adalah untuk mengatur pemisahan dan tata letak teks. Tag <hr> digunakan untuk membuat garis horizontal yang berfungsi sebagai pemisah antar bagian konten, sedangkan tag <br> digunakan untuk memindahkan teks ke baris baru tanpa membuat paragraf baru. Keduanya membantu memperjelas struktur dan tampilan halaman web.


<p>Kalimat pertama.<br/>Kalimat kedua di bawah.</p>
<hr/>
<p>Paragraf setelah garis pemisa</p>
