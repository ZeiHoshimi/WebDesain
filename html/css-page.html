‚ùì Materi 1: Apa Itu CSS?

CSS atau Cascading Style Sheets adalah alat yang digunakan untuk mengatur tampilan halaman web agar terlihat menarik. Jika HTML adalah kerangka atau tulang dari sebuah halaman, maka CSS adalah baju dan hiasannya. Dengan CSS, kita bisa mengubah warna tulisan, ukuran huruf, jenis font, jarak antar bagian, warna latar belakang, dan banyak lagi. Jadi, CSS membantu membuat halaman web tidak hanya berisi teks biasa, tapi juga enak dilihat dan mudah digunakan.

üìå Materi 2: KEGUNAAN CSS

CSS berfungsi untuk mengatur dan mempercantik tampilan halaman web dengan cara mengubah warna, ukuran, jenis huruf, posisi, dan layout elemen, serta membuat tampilan responsif dan menambahkan efek visual seperti animasi, sehingga halaman web menjadi lebih menarik, terstruktur, dan nyaman digunakan di berbagai perangkat.

Contoh CSS dasar


body {
  background-color: #f5f5f5;
  font-family: Arial, sans-serif;
}

h1 {
  color: #333;
  text-align: center;
}

button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
}
